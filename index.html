How to Use This Tool

Step 1: Configure API

Enter your API key in the designated field. Select or enter your API endpoint and model.

Step 2: Test Connection

Click the "Test API Key" button to validate your API credentials.

Step 3: Chat Interface

Once your API is validated, use the chat interface to interact with the model. Type messages and view responses.

Additional Features

  • Hide/show and copy your API key
  • Download your chat history
  • View and clear your test history
  • Toggle between light and dark mode
  • Share this tool with others

Security Note

Your API key is not stored on our servers. It is only used for direct API calls from your browser.
